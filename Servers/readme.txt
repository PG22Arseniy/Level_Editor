---------------------------------------
command line to run server: npm run server 
app is ruuning on: http://localhost:3400/  


# *** How to use ***
---------------------------------------
create level
1) drag objects to the scene
2) fill up level form
3) hit save under level form

load level
1) choose name from drop down list
2) hit load level
3) (You can edit it and hit save)

load object
1) choose name from drop down list
2) hit load object
3) (You can edit it and hit save) 

create object
2) fill up object form
3) hit save under level form
